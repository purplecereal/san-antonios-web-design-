The difference between bluehost and wordpress



The difference between bluehost and wordpress is that first and foremost bluehost is a hosting company and wordpress is a cms that provides hosting and also provides wordpress as a service.

If you use Bluehost as your hosting provider, you will have more bandwidth coverage, and you will be able to do more things with your website. Cpanel access is offered through WordPress as a service(wordpress.org). In contrast, WordPress hosted on wordpress.com does not provide Cpanel access. If you are going to be blogging, then wordpress.com would be fine. If you need something more complex where you will be offering services, you need wordpress.org and can host this via Bluehost. Directory sites, sites that need custom plugins, and other more complex sites will generally benefit from hosting their WordPress site on Bluehost.
